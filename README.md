# cool-crusts
lil test server, doesn't do much but is very :cool:

## Usage
This is a [module party](http://module.party/):
```sh
$ npm install
$ npm test
$ npm start
```

## Logs
To print the logs do:
```sh
$ npm run log <key printed by npm start>
```
